Algorithms for Selected Problems


1. Minimum of a List

Algorithm:

1. Input:List of numbers, `numList`

2. Initialization: Set `minNum` to the first element of `numList`

3. Processing Steps:

- For each number `num` in `numList`:

- If `num` is less than `minNum`:

- Set `minNum` to `num`

4. Output: `minNum`


2. Sum of Digits

Algorithm:

1. Input: A positive integer, `n`

2. Initialization: Set `sum` to 0

3. Processing Steps:

- While `n` is greater than 0:

- Add `n % 10` to `sum`

- Update `n` to `n // 10`

4. Output: `sum`



3. Factorial Calculation

Algorithm:

1. Input: A non-negative integer, `N`

2. Initialization: Set `factorial` to 1

3. Processing Steps:

- Selection: If `N` is 0 or 1:

- Return 1 (as the factorial of 0 and 1 is 1)

- Else:

- For each integer `i` from 1 to `N`:

- Multiply `factorial` by `i`

4. Output: `factorial`


Examples of Iteration and Selection:

1. Iteration:

- Minimum of a List: The for loop iterates over each element in `numList`.

- Sum of Digits: The while loop continues until `n` is greater than 0.

- Factorial Calculation: The for loop iterates from 1 to `N`.


2. Selection:

- Minimum of a List: The if statement checks if `num` is less than `minNum`.

- Sum of Digits: Implicit in ensuring `n` is greater than 0 to continue the while loop.

- Factorial Calculation: The if statement checks if `N` is 0 or 1 and returns 1 if true.
